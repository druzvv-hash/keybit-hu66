# Release checklist

- [ ] Public build contains no real customer photos or bitting pairs.
- [ ] EXIF and metadata reviewed for every public image.
- [ ] `npm test` passes.
- [ ] UI checked in Ukrainian and English.
- [ ] Mobile touch controls checked on Android.
- [ ] Automatic and manual alignment checked.
- [ ] Raw code and MACS candidate remain separately visible.
- [ ] High-confidence wrong-result regressions reviewed.
- [ ] `CHANGELOG.md` updated.
- [ ] Version updated in title, header, footer, debug export, and dist filename.
- [ ] Social preview and screenshots match the release.
- [ ] Release notes repeat the alpha/verification warning.
