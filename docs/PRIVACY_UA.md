# Приватність

## Які дані обробляє застосунок

KeyBit обробляє вибране користувачем фото та локальні налаштування інтерфейсу, наприклад мову й параметри вирівнювання.

## Де відбувається обробка

Декодування зображення, вирівнювання, виділення доріжки й оцінка коду виконуються безпосередньо в браузері на пристрої користувача.

## Чого немає в застосунку

- аналітики;
- телеметрії;
- рекламних SDK;
- endpoint для завантаження фото;
- віддаленого AI/ML-запиту;
- облікового запису;
- бази кодів виробників.

## Локальне збереження

Вибрана мова може зберігатися в `localStorage`, якщо це дозволяє поточний browser origin. Debug-зображення та JSON зберігаються лише після явної дії користувача.

## Примітка щодо хостингу

GitHub Pages або інший хостинг може вести звичайні журнали доступу. Після завантаження статичних файлів сам KeyBit не передає вибране фото на сервер.

## Безпека публічних issue

Не завантажуй у відкриті issue впізнавані фото ключів, відповідний реальний бітинг, VIN, адреси, документи клієнта або EXIF-геолокацію.
