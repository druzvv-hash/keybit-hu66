# Changelog

All notable public changes to KeyBit HU66 are documented here.

## [0.3.5-alpha] - 2026-07-21

### Added

- Complete Ukrainian/English UI switch.
- Local language persistence where the browser origin permits it.
- Public GitHub-ready build without private key photos or bitting pairs.
- Repository documentation, privacy policy, validation protocol, and contribution rules.

### Retained

- Automatic/manual vertical alignment.
- Two-point manual blade frame.
- HU66 stencil and position guides.
- Trajectory-based internal-track decoder.
- HU66 MACS validation.
- Debug image and JSON export.

### Fixed

- Corrected the documentation preview so HU66 positions are numbered from the shoulder toward the tip (1 = 3 mm, 8 = 24 mm).
- Added explicit position-numbering notes to the English and Ukrainian README files and the algorithm overview.

### Safety

- Public alpha warning remains visible.
- Results must be independently verified before cutting.

## [0.3.4-alpha]

- Added HU66 stencil.
- Added raw-code MACS validation and corrected candidate display.

## [0.3.3-alpha]

- Added independent vertical alignment stage and fine rotation controls.

## [0.3.2-alpha]

- Fixed Android touch handling for manual tip/shoulder markers.

## [0.3.1-alpha]

- Added unreliable-alignment rejection and manual two-point geometry.
